### Listas Personalizadas

<VideoEmbed url="https://www.tella.tv/video/cmcf9uhgc00070bl44zyi3gwg/view" />

O que são listas personalizadas?

{/* truncate */}

As listas personalizadas são uma nova funcionalidade que permite criar agrupamentos personalizados de atendimentos no iHelp, além das listas padrão como **Meus**, **Departamento** e **Novos**.

***

➕ Como criar uma lista personalizada?

1. Clique em **Adicionar Lista**.
2. Dê um nome à nova lista.
3. Clique em **Adicionar Filtro** para definir os critérios da sua lista.

***

🔍 Quais filtros posso usar?

Você pode combinar os seguintes filtros para personalizar sua lista:

* **Status**: Atendimentos encerrados ou em aberto.
* **Departamento**: Filtrar por departamentos existentes.
* **Usuário**: Selecionar atendimentos por usuário específico.
* **Canal**: Filtrar por número/canal de atendimento (Ex: API Oficial, número comercial etc).
* **Tag**: Ideal para destacar temporariamente atendimentos com uma tag específica.

***

🧪 Exemplo prático de uso

Foi criada uma lista filtrando o canal **API Oficial**. Após salvar, a lista mostra **todos os atendimentos realizados através deste canal**, **independente do usuário** que atendeu.

***

📚 Posso ter várias listas diferentes?

Sim! Você pode criar diversas listas para diferentes objetivos:

* Lista por número comercial
* Lista por canal específico
* Lista para um único usuário
* Lista com tags específicas

***

🎯 Para que servem as listas personalizadas?

* Facilitar a visualização de atendimentos específicos.
* Organizar melhor o fluxo de trabalho da equipe.
* Acompanhar canais, usuários ou situações temporárias com mais foco.
