### FAQ – Como montar uma campanha pela API Oficial

#### ❓ O que é uma campanha via API Oficial?

A campanha via API Oficial é muito parecida com a campanha comum, porém possui algumas regras específicas — e até mais simples — principalmente em relação à lista e às variáveis do template.

***

#### ❓ Como iniciar uma campanha pela API Oficial?

1. Clique em **Nova Campanha**
2. Dê um **nome** para a campanha
3. Clique em **Continuar**

***

#### ❓ Qual é a principal diferença na hora de subir a lista?

A principal diferença está na **estrutura da planilha**.

* Em campanhas comuns, a lista pode conter várias colunas/variáveis extras.
* Já na **API Oficial**, a lista **precisa ser exatamente igual ao template**.

📌 Exemplo:\
Se o template possui **2 variáveis**, a planilha deve conter:

* Número de telefone
* Variável 1
* Variável 2

⚠️ Não pode ter:

* Colunas a mais
* Colunas a menos

Caso contrário, a campanha retornará erro.

***

#### ❓ Posso editar as variáveis dentro da campanha?

Não.\
Diferente das campanhas comuns:

* As variáveis **não podem ser editadas**
* Elas já vêm automaticamente como **Variável 1, Variável 2, Variável 3**, conforme o template

***

#### ❓ O que acontece após subir a planilha?

Após subir a planilha:

1. Clique em **Continuar**
2. O sistema já mostra a lista carregada
3. Você seleciona o **canal**
4. Os **templates aparecem automaticamente** disponíveis para aquele canal

Assim, você já escolhe exatamente o template que será utilizado.

***

#### ❓ É necessário configurar variações de mensagem?

Não.\
Na campanha via API Oficial:

* Não existe configuração de variações
* Esse campo não pode ser editado

***

#### ❓ Como funciona o botão do template?

O botão já vem conforme configurado no template.\
Basta selecionar o **departamento** que receberá os retornos.

***

#### ❓ Posso disparar a campanha imediatamente?

Sim.

* Você pode **disparar na hora**
* Ou **programar para outro dia**, exatamente como nas campanhas comuns
* Não é necessário configurar período programado

***

#### ❓ A previsão de conclusão da campanha é confiável?

Nesse momento, **não totalmente**.

📌 Isso acontece porque:

* A API Oficial dispara **todas as mensagens de uma vez**
* Diferente da campanha comum, que dispara em lotes

Por isso, a previsão exibida pode não refletir o tempo real de envio.

***

#### ❓ O restante do processo muda?

Não.\
Após o resumo:

* O fluxo segue **igual ao de uma campanha comum**
* Revisar informações
* Confirmar
* Disparar
