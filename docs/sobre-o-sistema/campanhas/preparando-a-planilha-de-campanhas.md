#### 📝 Passo a passo: Como montar a planilha para sua campanha no iHelp

1. **Crie uma nova campanha**
   * No painel do iHelp, clique em **“Nova Campanha”**.
   * Dê um nome para sua campanha (ex: *“Salão – Corte Junho”*).
2. **Baixe o modelo de planilha**
   * Assim que a campanha for criada, o sistema vai disponibilizar uma **lista modelo**.
   * Clique para **baixar o arquivo**, é com essa planilha que você vai trabalhar.
3. **Preencha os números de telefone**
   * Na **primeira coluna**, insira os números de telefone **com DDD** (ex: `11999999999`).
   * ⚠️ **Atenção:** Sem DDD, a mensagem não será enviada. Para números internacionais, inclua o **DDI** (ex: `+55` para o Brasil).
4. **Adicione uma saudação**
   * Na **segunda coluna**, insira uma saudação, como: `Oi`, `Olá`, `Oie`.
   * Essa saudação pode ser combinada com o nome do cliente, se quiser personalizar:\
     Ex: **“Olá, Maria”**.
5. **Inclua variações de mensagem**
   * Adicione uma nova coluna com as variações da mensagem principal.
   * **Exemplo 1:** “Lembrei de você! Vamos agendar seu corte?”
   * **Exemplo 2:** “Vi que faz um tempo que você não aparece. Vamos marcar um horário?”
   * ✅ Quanto **mais variações**, **menor a chance de bloqueio ou marcação como spam**.
6. **Personalize com o nome do cliente (opcional)**
   * Se quiser chamar a pessoa pelo nome, inclua uma coluna com os nomes.
   * Exemplo de mensagem: **“Olá, Carla! Lembrei de você. Vamos agendar seu corte?”**
7. **Dicas finais**
   * Sempre termine a mensagem com uma **pergunta**, para estimular o cliente a responder.
   * Verifique se as mensagens fazem sentido quando combinadas *(saudação + nome + texto)*.
   * Salve a planilha em formato **`.xlsx`** e suba na sua campanha.
