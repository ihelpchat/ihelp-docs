---
title: "Deixarei de ter acesso a algo?"
---

Sim. A API Oficial é uma infraestrutura pensada para operações comerciais em escala — e por isso funciona de forma diferente do aplicativo convencional. Algumas funcionalidades do WhatsApp que estamos acostumados no dia a dia simplesmente não existem nessa versão.

Conhecer essas limitações antes da migração é importante para não ter surpresas e para planejar como sua operação vai se adaptar.

***

**O que não está disponível na API Oficial**

**Acesso pelo app ou WhatsApp Web** Após a migração, o número não pode mais ser aberto no aplicativo do WhatsApp — nem no celular, nem pelo WhatsApp Web. Todo o acesso passa a ser exclusivamente pela plataforma iHelpChat.

**Chamadas de voz e vídeo** A API não suporta ligações. Se o seu atendimento depende de chamadas pelo WhatsApp, será necessário manter um número separado no aplicativo convencional para essa finalidade.

**Status** Não é possível publicar nem visualizar stories/status pelo número da API. Essa funcionalidade é exclusiva do aplicativo.

**Grupos** O número da API não pode participar de grupos do WhatsApp — nem criar, nem ser adicionado.

**Foto de perfil do contato** Por limitação da própria API, não é possível visualizar a foto de perfil dos contatos com quem você conversa.

**Responder mensagem específica (citar)** A função de citar uma mensagem específica dentro de uma conversa não está disponível na API.

**Comunicação entre dois números WABA** Dois números que operam como API Oficial não conseguem trocar mensagens entre si diretamente. Se precisar de um canal interno de comunicação via WhatsApp, ele precisará ser feito por um número no aplicativo convencional.

***

**Vale a pena mesmo assim?**

Para operações comerciais, sim — na grande maioria dos casos.

As limitações acima afetam principalmente funcionalidades de uso pessoal. O que a API entrega em troca — estabilidade, escala, múltiplos atendentes, automação e segurança operacional — supera com folga o que se abre mão, especialmente para empresas que usam o WhatsApp como canal principal de vendas ou atendimento.

A pergunta certa não é "o que vou perder?", mas sim "o que o meu negócio precisa para crescer?". Se a resposta envolve volume, organização e continuidade, a API é o caminho.
